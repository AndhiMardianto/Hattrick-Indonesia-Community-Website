# Hattrick Indonesia Community Website

Salam Hattrickers.

Website komunitas **Hattrick Indonesia** yang dibangun untuk menyediakan berbagai informasi, artikel, panduan, dan konten seputar permainan **Hattrick** bagi para manajer di Indonesia.

Proyek ini dikembangkan dan dikelola oleh **The Blind Hattrick** sebagai bentuk kontribusi untuk komunitas Hattrick game manager di Indonesia.

## Fitur

* Artikel dan panduan Hattrick game manager.

## Teknologi

* Astro
* HTML
* CSS
* JavaScript
